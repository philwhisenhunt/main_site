The Complete Beginner's Guide to Learning to Code

Intro

  • Anyone can learn how to code. This is the guide I wish I had been able to access when I was learning.

First Steps

  • Start with a text editing program. On mac, that’s Text Edit. It is already installed, so just find it in your applications folder and open it up
  • Make a new document and type “Hello World”. Then save it, and name it index
  • Find that file wherever you saved it, and double click it. It will open in your default browser.
  • See the text in the top left corner? That is the most basic form of a website. The only difference is that a website is accessible from anywhere. And that is done by putting it on a computer that accessible from anywhere, called a server.
  • To make a header, wrap the text in a h1 tag:
    <h1>Header</h1>
  • To make a paragraph, wrap it in a p tag:
    <p>I have a sentence here. And another sentence after it.</p>
